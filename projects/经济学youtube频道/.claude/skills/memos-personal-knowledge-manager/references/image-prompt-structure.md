# 经济学视频配图 Prompt 结构模板

## 使用方法

Prompt 由三部分拼接而成，顺序固定：

**内容描述** → **风格模块** → **质量后缀**

---

## 内容描述

描述画面内容（英文，详细）：
- 主体：人物、物体、场景
- 动作：正在做什么
- 象征元素：与主题相关的隐喻符号
- 构图：布局、视角、平衡感

---
